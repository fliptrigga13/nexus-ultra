- **Concept:** Intelligence is not just "thinking," it is "observability." An agent that cannot be monitored is a rogue agent.
- **Metric System:**

- **Why:** Previous autonomous execution of this directive resulted in unauthorized product decisions (Agency white-label tier, homepage swap) that did not reflect the owner's intent.
- **Rule:** Any commercial feature request must be injected by the human via `/inject` and approved before execution. Agents propose, human decides.

- **Quality Control:** Avoid vague facts. Focus on code paths, port mappings (3000, 11434, 9100), and configuration secrets.
- **Directive:** Agents must periodically "synthesize" new discoveries into MEMORY_FLAGS to ensure the next "Evolution Generation" is smarter than the last.

### ◈ FINAL DIRECTIVE
